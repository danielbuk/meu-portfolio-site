Análise de Preços e Operações de Voos na Índia

Um projeto de portfólio de Daniel Rocha

Introdução

Olá! Meu nome é Daniel Rocha e sou Analista de Dados com experiência no setor de logística e transporte. Neste projeto, realizei uma análise exploratória completa de um conjunto de dados contendo mais de 300.000 voos comerciais na Índia.

Objetivo Principal: Extrair insights acionáveis sobre a dinâmica de precificação, a performance das companhias aéreas e os fatores que mais influenciam o custo e a duração das viagens. Como profissional da área de logística, entendo que a otimização de rotas e a compreensão de custos são fundamentais para a eficiência operacional, e apliquei essa perspectiva na minha análise.

Ferramentas Utilizadas: Python, Pandas, Matplotlib e Seaborn.

Análise Exploratória e Insights

1. Como os preços das passagens se distribuem?

Para entender o cenário geral de precificação, comecei analisando a distribuição dos preços.

Insights: O gráfico mostra claramente dois picos. O primeiro, mais alto, corresponde aos voos da classe econômica (maioria na faixa de ₹5.000 a ₹15.000). O segundo pico, por volta de ₹40.000 a ₹60.000, indica a precificação da classe executiva. Essa bimodalidade mostra a importância de segmentar a análise por classe.

Código Python para esta Análise:

def create_dist_precos(df):
    fig = plt.figure(figsize=(12, 6))
    sns.histplot(df['price'], kde=True, bins=50, color='royalblue')
    plt.title('Distribuição dos Preços das Passagens (em Rúpia Indiana)', fontsize=16)
    plt.xlabel('Preço (INR)', fontsize=12)
    plt.ylabel('Frequência', fontsize=12)
    return plot_to_base64(fig)

2. Qual o market share e a estratégia de preços de cada companhia?

Investiguei a participação de cada companhia no volume total de voos e como isso se relaciona com sua estratégia de preços. Os dois gráficos abaixo, juntos, contam essa história.

Insights: A Vistara domina em volume de voos e tem o preço médio mais alto, focando no segmento premium. Em contraste, AirAsia e Indigo são as líderes de baixo custo. A Air India compete diretamente com a Vistara no segmento de serviço completo.

Código Python para esta Análise:

def create_voos_por_companhia(df):
    fig = plt.figure(figsize=(12, 7))
    order = df['airline'].value_counts().index
    ax = sns.countplot(y='airline', data=df, order=order, palette='viridis')
    ax.set_title('Número de Voos por Companhia Aérea', fontsize=16)
    ax.set_xlabel('Contagem de Voos', fontsize=12)
    ax.set_ylabel('Companhia Aérea', fontsize=12)
    for p in ax.patches:
        width = p.get_width()
        ax.text(width + 1000, p.get_y() + p.get_height() / 2., f'{int(width)}', va='center')
    return plot_to_base64(fig)

def create_preco_medio_por_companhia(df):
    fig = plt.figure(figsize=(12, 7))
    preco_medio_cia = df.groupby('airline')['price'].mean().sort_values(ascending=False)
    ax = sns.barplot(y=preco_medio_cia.index, x=preco_medio_cia.values, palette='plasma')
    ax.set_title('Preço Médio da Passagem por Companhia Aérea', fontsize=16)
    ax.set_xlabel('Preço Médio (INR)', fontsize=12)
    ax.set_ylabel('Companhia Aérea', fontsize=12)
    for p in ax.patches:
        width = p.get_width()
        ax.text(width + 500, p.get_y() + p.get_height() / 2., f'₹{width:,.0f}', va='center')
    return plot_to_base64(fig)

3. Qual a proporção de voos por classe?

Verifiquei a distribuição dos assentos ofertados entre as classes Econômica e Executiva.

Insights: Cerca de 69% dos voos no dataset são da classe econômica, o que é esperado. No entanto, a classe executiva representa uma fatia significativa de 31%, impulsionada pelas companhias premium.

Código Python para esta Análise:

def create_voos_por_classe(df):

    ax.set_ylabel('Contagem de Voos', fontsize=12)

        height = p.get_height()

4. O quão antes devo comprar minha passagem?

Analisei a variação do preço médio de acordo com o número de dias restantes para o voo.

Insights Estratégico: Os preços são estáveis quando a compra é feita com mais de 20 dias de antecedência. A partir daí, sobem, com um aumento exponencial nos últimos 5 dias. Para otimização de custos, a recomendação é clara: comprar com no mínimo três semanas de antecedência.

Código Python para esta Análise:

def create_preco_vs_dias_restantes(df):
    fig = plt.figure(figsize=(12, 6))
    preco_por_antecedencia = df.groupby('days_left')['price'].mean().reset_index()
    sns.lineplot(x='days_left', y='price', data=preco_por_antecedencia, color='green')
    plt.title('Preço Médio da Passagem vs. Antecedência da Compra', fontsize=16)
    plt.xlabel('Dias Restantes para o Voo', fontsize=12)
    plt.ylabel('Preço Médio (INR)', fontsize=12)
    plt.grid(True)
    return plot_to_base64(fig)

Analisei a dispersão dos preços entre as classes para entender a faixa de valores.

Insights: Praticamente não há sobreposição de preços. O voo mais caro da classe econômica é quase sempre mais barato que o voo mais barato da classe executiva. A classe executiva tem preços mais consistentes (caixa do boxplot compacta), enquanto a econômica tem uma variação muito maior.

6. Como a duração dos voos e o número de paradas se comportam?

Fatores operacionais como duração e paradas são cruciais na logística.

Insights: Voos diretos duram em média 2-3 horas. Com uma parada, a duração mediana salta para 10-12 horas. Isso mostra o claro trade-off entre tempo e custo que o passageiro ou a empresa precisa considerar.

Código Python para esta Análise:

def create_dist_duracao(df):

7. Quais fatores numéricos mais impactam o preço?

Para quantificar as relações entre as variáveis, criei um heatmap de correlação.

Insights: A correlação mais forte é entre `price` e `duration` (0.20), indicando que voos mais longos tendem a ser mais caros. A correlação entre preço e antecedência (`days_left`) é fraca (-0.09) porque a relação não é linear, reforçando a importância de visualizar os dados além dos números de correlação.

Código Python para esta Análise:

def create_heatmap_correlacao(df):
    fig = plt.figure(figsize=(10, 8))
    numerical_cols = df.select_dtypes(include=np.number).columns
    correlation_matrix = df[numerical_cols].corr()
    sns.heatmap(correlation_matrix, annot=True, cmap='coolwarm', fmt=".2f", linewidths=.5)
    plt.title('Heatmap de Correlação entre Variáveis Numéricas', fontsize=16)
    return plot_to_base64(fig)

8. Quais são as rotas mais movimentadas e seus preços médios?

Finalmente, foquei nas rotas para entender a dinâmica dos mercados mais importantes.

Insights de Logística: As rotas partindo de Bangalore e a rota Mumbai -> Kolkata se destacam com os preços médios mais elevados. Para uma empresa com operações nessas cidades, a análise de preços por rota é fundamental para o planejamento de viagens e negociação com companhias aéreas.

Código Python para esta Análise:

def create_preco_medio_top10_rotas(df):
    fig = plt.figure(figsize=(16, 9))
    df['route'] = df['source_city'] + ' -> ' + df['destination_city']
    top_10_routes = df['route'].value_counts().nlargest(10).index
    df_top_routes = df[df['route'].isin(top_10_routes)]
    ax = sns.barplot(x='route', y='price', data=df_top_routes, order=top_10_routes, palette='Set2', errorbar=None)
    ax.set_title('Preço Médio nas 10 Rotas Mais Populares', fontsize=16)
    ax.set_xlabel('Rota', fontsize=12)
    ax.set_ylabel('Preço Médio (INR)', fontsize=12)
    plt.xticks(rotation=45, ha='right')
    for p in ax.patches:
        height = p.get_height()
        ax.text(p.get_x() + p.get_width() / 2., height + 200, f'₹{height:,.0f}', ha='center')
    return plot_to_base64(fig)
